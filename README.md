# MenteViva

Estimulador cognitivo IoT para idosos — BitDogLab v6.3 | EmbarcaTech

## Sobre

Dispositivo embarcado que oferece exercícios cognitivos interativos (memória e reflexo) para idosos, acompanhando a evolução do desempenho ao longo do tempo.

## Hardware

- BitDogLab v6.3 (Raspberry Pi Pico W / RP2040)
- Nenhum componente externo necessário

## Como compilar

```bash
mkdir build && cd build
cmake .. -DPICO_BOARD=pico_w
make -j4
```
